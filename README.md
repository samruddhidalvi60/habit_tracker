# 📊 Habit Tracker with Streak System

A simple Python project to track daily habits and maintain streaks.

## 🚀 Features
- Add new habits
- Mark habits as completed
- Calculate current streak
- Track longest streak
- Store data using file handling

## 🛠️ Technologies Used
- Python
- File Handling
- datetime module

## 💡 How it Works
- Each habit stores completed dates
- Streak is calculated using consecutive days logic
- Data is saved in a text file

## ▶️ How to Run
1. Install Python
2. Run the program:
   python habit_tracker.py
